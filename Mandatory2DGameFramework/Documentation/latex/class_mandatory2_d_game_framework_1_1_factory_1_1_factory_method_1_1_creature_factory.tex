\doxysubsection{Detailed Description}
\doxylink{namespace_mandatory2_d_game_framework_1_1_factory}{Factory} for creating creatures with appropriate equipment. Provides a centralized way to instantiate creatures. 

Uses a switch-\/based implementation to create different creature types. Each creature type is created with class-\/appropriate starting equipment using item factories (\doxylink{class_mandatory2_d_game_framework_1_1_factory_1_1_abstract_factory_1_1_concrete_factories_1_1_warrior_item_factory}{Warrior\+Item\+Factory}, \doxylink{class_mandatory2_d_game_framework_1_1_factory_1_1_abstract_factory_1_1_concrete_factories_1_1_mage_item_factory}{Mage\+Item\+Factory}, \doxylink{class_mandatory2_d_game_framework_1_1_factory_1_1_abstract_factory_1_1_concrete_factories_1_1_hunter_item_factory}{Hunter\+Item\+Factory}).
