\doxysubsection{Detailed Description}
\doxylink{namespace_mandatory2_d_game_framework_1_1_factory}{Factory} for creating Hunter-\/themed equipment. Creates medium Leather armor and ranged Gun weapons. 

This factory produces a family of items suitable for Hunter class characters\+: 
\begin{DoxyItemize}
\item {\bfseries{Weapon\+:}} Gun (moderate damage, long range) 
\item {\bfseries{Armor\+:}} Leather material (medium defense values, balanced) 
\end{DoxyItemize}

Default values\+: Weapon damage 60, armor defense ranges from 5-\/15 depending on slot.
