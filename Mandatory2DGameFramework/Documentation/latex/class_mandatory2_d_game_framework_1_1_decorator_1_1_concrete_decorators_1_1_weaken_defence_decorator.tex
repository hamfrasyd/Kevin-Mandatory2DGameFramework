\doxysubsection{Detailed Description}
Concrete decorator that weakens defense value. Decreases the defense value of a wrapped defense item. 

This decorator wraps a defense item and decreases its defense value by Weaken\+Amount (default is 3). The defense value cannot go below zero -\/ negative values are clamped to 0. 

Useful for representing damaged, cursed, or corroded armor.
