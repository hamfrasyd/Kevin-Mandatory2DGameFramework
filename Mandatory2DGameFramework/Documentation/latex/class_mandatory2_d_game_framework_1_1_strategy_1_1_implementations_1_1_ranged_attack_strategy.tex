\doxysubsection{Detailed Description}
Ranged attack strategy that provides bonus damage at long distances. Designed for long-\/range combat. 

This strategy is designed for ranged attackers and rewards positioning\+: 
\begin{DoxyItemize}
\item {\bfseries{Short/\+Medium range (0-\/19)\+:}} Uses \doxylink{interface_mandatory2_d_game_framework_1_1_interfaces_1_1_i_weapon_aa29fdf012ddc8ddf6db0d0f12dfa114d}{IWeapon.\+Get\+Damage} or default base damage if unarmed 
\item {\bfseries{Long range (20+)\+:}} Applies 50\% damage bonus for optimal positioning 
\end{DoxyItemize}

Used by default by \doxylink{class_mandatory2_d_game_framework_1_1_template_1_1_implementations_1_1_mage}{Mage} and \doxylink{class_mandatory2_d_game_framework_1_1_template_1_1_implementations_1_1_hunter}{Hunter} classes.
