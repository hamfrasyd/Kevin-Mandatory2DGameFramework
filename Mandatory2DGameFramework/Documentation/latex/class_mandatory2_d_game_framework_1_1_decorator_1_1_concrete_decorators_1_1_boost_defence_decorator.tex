\doxysubsection{Detailed Description}
Concrete decorator that boosts defense value. Increases the defense value of a wrapped defense item. 

This decorator wraps a defense item and increases its defense value by Boost\+Amount. The boost can be applied to any \doxylink{interface_mandatory2_d_game_framework_1_1_decorator_1_1_interfaces_1_1_i_defence_item}{IDefence\+Item}, including other decorated items, allowing for decorator chaining.
