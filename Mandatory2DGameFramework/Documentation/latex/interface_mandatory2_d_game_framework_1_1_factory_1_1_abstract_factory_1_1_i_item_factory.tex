\doxysubsection{Detailed Description}
Interface for creating families of related game items. Each concrete factory creates items that work well together thematically and mechanically. 

Factories return product interfaces (\doxylink{interface_mandatory2_d_game_framework_1_1_interfaces_1_1_i_weapon}{IWeapon}, \doxylink{interface_mandatory2_d_game_framework_1_1_decorator_1_1_interfaces_1_1_i_defence_item}{IDefence\+Item}) instead of concrete classes, ensuring item families are consistent (Warriors get Plate armor, Mages get Cloth, etc.) and providing a complete set of equipment through a single factory.
