\doxysubsection{Detailed Description}
Manages equipped armor pieces as a collection. Can contain both individual armor pieces and other armor sets. 

Both \doxylink{class_mandatory2_d_game_framework_1_1_core_1_1_items_1_1_defence_item}{Defence\+Item} and \doxylink{class_mandatory2_d_game_framework_1_1_composite_1_1_implementations_1_1_equipped_armor_set}{Equipped\+Armor\+Set} implement \doxylink{interface_mandatory2_d_game_framework_1_1_composite_1_1_interfaces_1_1_i_game_item}{IGame\+Item}, allowing them to be treated uniformly. 

Uses recursive aggregation to calculate total values from nested structures. This allows for nested armor sets and flexible armor organization.
