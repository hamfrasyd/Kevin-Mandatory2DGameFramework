\doxysubsection{Detailed Description}
\doxylink{namespace_mandatory2_d_game_framework_1_1_factory}{Factory} for creating Mage-\/themed equipment. Creates light Cloth armor and magical Staff weapons. 

This factory produces a family of items suitable for Mage class characters\+: 
\begin{DoxyItemize}
\item {\bfseries{Weapon\+:}} Staff (moderate damage, short-\/medium range) 
\item {\bfseries{Armor\+:}} Cloth material (lowest defense values, emphasizes mobility) 
\end{DoxyItemize}

Default values\+: Weapon damage 76, armor defense ranges from 3-\/12 depending on slot.
