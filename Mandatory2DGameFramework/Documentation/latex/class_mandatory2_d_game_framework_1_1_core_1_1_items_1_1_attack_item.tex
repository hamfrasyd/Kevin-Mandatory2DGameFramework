\doxysubsection{Detailed Description}
Represents an attack item/weapon in the game. Implements \doxylink{interface_mandatory2_d_game_framework_1_1_interfaces_1_1_i_weapon}{IWeapon} interface. 

Weapons are created by item factories and can be equipped by creatures. Supports operator overloading (+) for dual wielding functionality.
