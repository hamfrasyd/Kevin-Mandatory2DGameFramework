\doxysubsection{Detailed Description}
Melee attack strategy that provides bonus damage when attacker\textquotesingle{}s HP is low. Designed for close-\/range combat. 

This strategy is designed for melee fighters and provides a "{}berserker"{} mechanic\+: 
\begin{DoxyItemize}
\item {\bfseries{Normal\+:}} Uses \doxylink{interface_mandatory2_d_game_framework_1_1_interfaces_1_1_i_weapon_aa29fdf012ddc8ddf6db0d0f12dfa114d}{IWeapon.\+Get\+Damage} or default base damage if unarmed 
\item {\bfseries{Low HP (below 50\%)\+:}} Applies 50\% damage bonus 
\end{DoxyItemize}

Used by default by \doxylink{class_mandatory2_d_game_framework_1_1_template_1_1_implementations_1_1_warrior}{Warrior} class.
