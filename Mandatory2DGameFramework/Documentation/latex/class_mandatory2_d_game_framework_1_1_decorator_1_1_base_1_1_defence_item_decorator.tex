\doxysubsection{Detailed Description}
\doxylink{namespace_mandatory2_d_game_framework_1_1_decorator_1_1_base}{Base} decorator class for defense items. Wraps an \doxylink{interface_mandatory2_d_game_framework_1_1_decorator_1_1_interfaces_1_1_i_defence_item}{IDefence\+Item} and delegates to it while allowing subclasses to modify behavior. 

This abstract class serves as the foundation for concrete decorators that modify defense item properties (boost or weaken defense values). 

Concrete decorators override the virtual properties to modify values while maintaining the wrapped item\textquotesingle{}s identity.
