\doxysubsection{Detailed Description}
Interface for objects that can be attacked and take damage. Implemented by creatures and other damageable entities in the game world. 

This interface enables polymorphic targeting in combat. Any object implementing \doxylink{interface_mandatory2_d_game_framework_1_1_interfaces_1_1_i_attackable}{IAttackable} can be the target of an attack, allowing for flexible combat mechanics.
