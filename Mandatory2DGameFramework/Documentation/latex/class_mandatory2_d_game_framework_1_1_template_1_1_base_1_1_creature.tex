\doxysubsection{Detailed Description}
Abstract base class for all creatures in the game. Provides combat, equipment, and event notification capabilities. 

Creatures can\+: 
\begin{DoxyItemize}
\item Perform attacks using attack strategies 
\item Equip \doxylink{interface_mandatory2_d_game_framework_1_1_interfaces_1_1_i_weapon}{IWeapon} weapons and armor 
\item Notify observers when taking damage or dying 
\item Override attack modifiers for class-\/specific bonuses 
\end{DoxyItemize}
