Detailed Description

Represents an attack item/weapon in the game. Implements IWeapon interface.

Weapons are created by item factories and can be equipped by creatures. Supports operator overloading (+) for dual wielding functionality.

// Create a weapon
AttackItem sword = new AttackItem("Excalibur", WeaponType.Sword, 50, 1);
// Dual wield
AttackItem dagger = new AttackItem("Dagger", WeaponType.Dagger, 10, 1);
AttackItem dualWield = sword + dagger; // Combined: 60 damage

Combines two weapons for dual wielding using the + operator.

Parameters
leftThe first weapon (main hand).
rightThe second weapon (off hand).
Returns
A new AttackItem with combined damage and maximum range. Returns non-null weapon if only one is provided, or empty hands if both are null.

Demonstrates operator overload requirement.

Combined weapon inherits the type from the left weapon.

Damage is additive, range uses the maximum of the two weapons.

AttackItem sword = new AttackItem("Sword", WeaponType.Sword, 30, 1);
AttackItem dagger = new AttackItem("Dagger", WeaponType.Dagger, 15, 1);
AttackItem dualWield = sword + dagger;
// Result: "Sword &amp; Dagger" with 45 damage
