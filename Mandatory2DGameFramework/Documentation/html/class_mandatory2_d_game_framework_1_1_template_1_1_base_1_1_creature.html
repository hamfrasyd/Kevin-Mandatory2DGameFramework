Detailed Description

Abstract base class for all creatures in the game. Provides combat, equipment, and event notification capabilities.

Creatures can:

  • Perform attacks using attack strategies
  • Equip IWeapon weapons and armor
  • Notify observers when taking damage or dying
  • Override attack modifiers for class-specific bonuses
// Create creature
Creature warrior = new Warrior("Bob", 1000);
// Observer pattern - attach logger (singleton, auto-starts combat logging)
warrior.Attach(CombatLogger.Instance);
// Equip items
warrior.EquipWeapon(new AttackItem("Sword", WeaponType.Sword, 30, 1));
warrior.EquippedArmor.Add(new DefenceItem("Helmet", ArmorSlot.Head, ArmorType.Plate, 10));
// Strategy + Template Method pattern - perform attack
Creature enemy = new Mage("Evil Mage", 800);
warrior.PerformAttack(enemy, range: 1);

int range = 0 )

Performs an attack on the target.

Parameters
targetThe target to attack.
rangeThe distance to the target. Default is 0 for melee range.

Attack algorithm:

  1. Validate attacker and target are alive
  2. Calculate base damage using attack strategy
  3. Apply class-specific modifiers via ApplyAttackModifiers
  4. Apply damage to target
Creature warrior = new Warrior("Bob", 1000);
Creature enemy = new Mage("Alice", 800);
// Melee attack
warrior.PerformAttack(enemy, range: 1);
// Ranged attack
enemy.PerformAttack(warrior, range: 25);
