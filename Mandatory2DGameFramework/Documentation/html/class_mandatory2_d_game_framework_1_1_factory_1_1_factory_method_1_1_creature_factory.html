Detailed Description

Factory for creating creatures with appropriate equipment. Provides a centralized way to instantiate creatures.

Uses a switch-based implementation to create different creature types. Each creature type is created with class-appropriate starting equipment using item factories (WarriorItemFactory, MageItemFactory, HunterItemFactory).

Starting attributes by class:

Class HP / Weapon / Armor Type
Warrior 1000 HP / Sword / Plate armor
Mage 800 HP / Staff / Cloth armor
Hunter 900 HP / Gun / Leather armor
// Create a warrior
Creature warrior = CreatureFactory.CreateCreature(ClassType.Warrior, "Bob");
// Bob is created with 1000 HP and a Sword weapon
// Create a mage
Creature mage = CreatureFactory.CreateCreature(ClassType.Mage, "Alice");
// Alice is created with 800 HP and a Staff weapon
// Combat
warrior.PerformAttack(mage, range: 1);
