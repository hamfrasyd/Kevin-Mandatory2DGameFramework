Detailed Description

Manages the game world including boundaries, objects, and creatures. Provides centralized management for all entities in the game.

The World class maintains collections of world objects and creatures, and enforces world boundaries through validation.

World world = new World(100, 100, "Fantasy World");
Creature warrior = new Warrior("Bob", 1000);
world.AddCreature(warrior);
List<Creature> allCreatures = world.GetCreatures();
