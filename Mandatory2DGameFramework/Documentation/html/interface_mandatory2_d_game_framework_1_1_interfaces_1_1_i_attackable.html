Detailed Description

Interface for objects that can be attacked and take damage. Implemented by creatures and other damageable entities in the game world.

This interface enables polymorphic targeting in combat. Any object implementing IAttackable can be the target of an attack, allowing for flexible combat mechanics.

IAttackable target = new Warrior("Enemy", 1000);
warrior.PerformAttack(target, range: 5);
if (!target.IsAlive)
{
Console.WriteLine("Target defeated!");
}
