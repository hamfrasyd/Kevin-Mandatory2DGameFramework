◆ ArmorSlot

enum Mandatory2DGameFramework.Enums.ArmorSlot

Defines the equipment slots where armor can be equipped on a creature. Each creature can have one armor piece per slot.

Used in conjunction with ArmorType to fully define an armor piece. ArmorSlot determines where the armor is worn, while ArmorType determines the material.

DefenceItem helmet = new DefenceItem("Iron Helmet", ArmorSlot.Head, ArmorType.Plate, 10);
creature.EquippedArmor.Add(helmet);

◆ ArmorType

enum Mandatory2DGameFramework.Enums.ArmorType

Defines the material types for armor pieces. Different materials provide different levels of protection and weight.

Material types are associated with creature classes through item factories:

  • Plate: Heavy armor for Warriors - highest defense
  • Leather: Medium armor for Hunters - balanced defense
  • Cloth: Light armor for Mages - lowest defense
// Warriors use Plate armor
IItemFactory warriorFactory = new WarriorItemFactory();
IDefenceItem plateHelmet = warriorFactory.CreateHelmet("Iron Helmet");
// Creates: ArmorType.Plate
// Mages use Cloth armor
IItemFactory mageFactory = new MageItemFactory();
IDefenceItem clothHelmet = mageFactory.CreateHelmet("Silk Hood");
// Creates: ArmorType.Cloth

◆ ClassType

enum Mandatory2DGameFramework.Enums.ClassType

Defines the types of creature classes available in the game. Used by CreatureFactory to create different creature types.

Each class type has unique characteristics:

Class Characteristics
Warrior Melee fighter with high HP (1000), uses Plate armor and Swords
Mage Ranged caster with medium HP (800), uses Cloth armor and Staves
Hunter Ranged attacker with medium-high HP (900), uses Leather armor and Guns
Creature warrior = CreatureFactory.CreateCreature(ClassType.Warrior, "Bob");
Creature mage = CreatureFactory.CreateCreature(ClassType.Mage, "Alice");

◆ WeaponType

enum Mandatory2DGameFramework.Enums.WeaponType

Defines the types of weapons available in the game. Different weapon types have varying damage, range, and characteristics.

Weapons are categorized by their attack style:

  • Melee weapons: Sword, Axe, Mace, Dagger (Range: 1)
  • Ranged weapons: Bow, Gun, Wand, Staff (Range: 2-30)
  • Unarmed: Default when no weapon equipped (Range: 1, low damage)
AttackItem sword = new AttackItem("Excalibur", WeaponType.Sword, 50, 1);
AttackItem gun = new AttackItem("Rifle", WeaponType.Gun, 20, 10);
