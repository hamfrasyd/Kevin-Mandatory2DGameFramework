Detailed Description

Represents a defensive item/armor in the game. Implements IDefenceItem and IGameItem interfaces.

Armor pieces can be:

  • Wrapped with decorators to modify defense values
  • Added to EquippedArmorSet to create armor collections
// Create armor
DefenceItem helmet = new DefenceItem("Iron Helmet", ArmorSlot.Head, ArmorType.Plate, 10);
// Use with Decorator
IDefenceItem boosted = new BoostDefenceDecorator(helmet);
int defense = boosted.Defense; // 15 (10 + 5)
// Use with Composite
creature.EquippedArmor.Add(helmet);
