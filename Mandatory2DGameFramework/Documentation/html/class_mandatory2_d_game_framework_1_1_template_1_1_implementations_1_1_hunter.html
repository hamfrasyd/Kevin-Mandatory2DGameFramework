Detailed Description

Hunter creature class - ranged attacker with medium-high HP and medium armor. Uses default attack modifier implementation (no class-specific bonuses).

Class Characteristics:

  • HP: 900 (between Warrior and Mage)
  • Strategy: Uses RangedAttackStrategy for long-range attacks
  • Armor: Leather material (medium defense, balanced)
  • Weapon: Gun (moderate damage, very long range)
  • Special: No class-specific modifiers, relies on strategy bonuses
// Create hunter
Hunter hunter = new Hunter("Robin");
// Equip medium gear
IItemFactory factory = new HunterItemFactory();
hunter.EquipWeapon(factory.CreateWeapon("Hunting Rifle"));
hunter.EquippedArmor.Add(factory.CreateChestArmor("Leather Tunic"));
// Long range attack with strategy bonus
hunter.PerformAttack(enemy, range: 30); // +50% from RangedAttackStrategy
