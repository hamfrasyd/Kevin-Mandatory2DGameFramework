Detailed Description

Interface for receiving notifications about creature events. Enables loose coupling between creatures and event handlers.

Observers are notified when creatures take damage or die. Multiple observers can be attached to a single creature, enabling features like:

  • Combat logging
  • Statistics tracking
  • UI updates
  • Achievement systems
// Create an observer (singleton, auto-starts combat logging)
ICreatureObserver logger = CombatLogger.Instance;
// Attach to creature
Creature warrior = new Warrior("Bob", 1000);
warrior.Attach(logger);
// Events trigger notifications
warrior.TakeDamage(100); // Triggers OnCreatureHit
warrior.TakeDamage(1000); // Triggers OnCreatureDied
