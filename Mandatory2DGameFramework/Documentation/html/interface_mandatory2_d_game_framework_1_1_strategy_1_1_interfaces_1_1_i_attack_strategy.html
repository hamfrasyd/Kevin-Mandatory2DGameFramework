Detailed Description

Interface for attack damage calculations. Enables different attack algorithms that can be swapped at runtime.

Different strategies provide different combat mechanics:

  • MeleeAttackStrategy: Bonus damage when HP < 50%
  • RangedAttackStrategy: Bonus damage when range â‰¥ 20

Strategies are set via Creature.SetAttackStrategy and used in Creature.PerformAttack.

// Create creature with strategy
Creature warrior = new Warrior("Bob", 1000);
// Warrior uses MeleeAttackStrategy by default
// Change strategy at runtime
warrior.SetAttackStrategy(new RangedAttackStrategy());
// Attack uses the current strategy
warrior.PerformAttack(enemy, range: 25);
